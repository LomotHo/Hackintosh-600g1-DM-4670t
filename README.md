# Hackintosh-600g1-DM-4670t

CLOVER版本：5120 | MacOS版本: 10.15.6

## 配置

 - CPU: i5 4670t
 - GPU: HD Graphics 4600
 - 无线网卡: 暂无
 - 硬盘: WD sn720

## BIOS设置

 - CFG lock -> unlock
 - DVMT -> 96M
这两个设置在bios设置界面是没有的需要用setup_var工具，此处暂时先不介绍
